Como desplegar el servidor en Render

1
Sube los archivos a GitHub

Crea un repo nuevo en GitHub llamado lighter-server y sube los archivos main.py, requirements.txt y render.yaml que te ha dado Claude.

2
Crea un Web Service en Render

Ve a render.com → New → Web Service → conecta tu repo lighter-server.

3
Configura la variable de entorno

En Render → Environment → Add variable:
LIGHTER_TOKEN = tu token ro:123:all:...

4
Deploy y copia la URL

Render te da una URL tipo https://lighter-tracker-server.onrender.com. Pegala arriba.

Como funciona
El servidor Python corre en Render 24/7, conectado al WebSocket de Lighter. Cada vez que cierras una operacion, recibe el trade con el P&L real al instante. Esta web consulta el servidor cada 10 segundos y muestra los datos actualizados desde cualquier dispositivo.
